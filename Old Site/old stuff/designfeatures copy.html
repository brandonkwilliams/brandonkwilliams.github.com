Guitar Case - Design Features

Spring/Summer 2012
All-State Industries Inc., Des Moines, Iowa

Logo Shell

Recessed Handle

  • Logo shell idea was decided upon from initial design sketches and models
  • Prominently features the Gibson USA logo script
  • Allows the passers-by to know the content of the case
  • Clients had an idea for a case featuring a recessed handle
  • This required a new, more complex geometry for the bottom shell of the case
  • Final design features a handle that lays flush in the side of the case when not in use

Custom Foam Padding

Tuning Station Accessory

  • Custom foam padding inserts were designed based on the geometry of the guitar itself
  • Inserts were machined from solid foam pieces using CNC and CAM software
  • Final interior of case covered with faux fur lining
  • Idea for an in-case device that allows the headstock of the guitar to be lifted out of the case for tuning
  • Entire box can be removed for easy access to necessary accessories
