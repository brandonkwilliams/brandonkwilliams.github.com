Portfolio Website

Winter Qtr. 2013
DSGN 370

Research/Brainstorm

Design Layouts

  • I researched different styles of web design by perusing several different inspiraion sites
  • This allowed me to be able to determine aspects of websites that I liked or didn't like and incorporate these ideas into my design
  • I created several iterations of possible layouts in Pixelmator using different colors, fonts, and navigation styles
  • After receiving feedback from several sources, I was able to develop a final design approach

Select Platform

Final Design

  • After several attempts, I selected the Twitter Bootstrap Framework on which to build my site
  • I had to read the documentation and experiment until I understood how the framework behaved
  • Finally, I was able to start putting together the website you see now
  • The final product is the website you are currently viewing
  • The website will continue to be updated and upgraded and has become a personal project to be carried on
  • The skills that I have learned over the course of these ten weeks will allow me to create websites again in the future

I was able to learn enough HTML5 and CSS3 to be able to build an entire website from scratch on the Twitter Bootstrap Framework in less than ten weeks. I feel that this is another example of my drive to learn any skill that allows me to create, and the web is another media in which I can now develop my ideas.
